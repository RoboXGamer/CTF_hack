RSA Common Factor – Write-Up
Overview
This challenge involved analyzing two RSA public keys that were generated for a dual-key escrow system. A message was encrypted using the first key, and both public keys along with the ciphertext were provided. The goal was to recover the original plaintext.

Steps
Examine the Provided Data
The challenge provided:

Two RSA public keys (modulus values (n_1) and (n_2))
The public exponent
A ciphertext encrypted using the first key.

Check for Weak Key Generation
Since both keys were generated by the same system, the possibility of a shared prime factor was considered.

Compute the GCD
The Greatest Common Divisor (GCD) of the two moduli (n_1) and (n_2) was calculated.
This revealed a common prime factor, meaning the two RSA keys were not independent.
